Help

To get started:

  1. Select one or many models
  2. Select one or many policy rules
  3. Optionally select shocks and output variables
  4. Hit 'Compare'

Hints:

  • Not all model and policy rule combinations are compatible, this is indicated by a shaded unavailable state
  • Models which are unavailable for the current selection of shocks/prules/outputs are shaded gray
  • Hover over any title/name to see a description of that object
